# Speech Emotion Recognition Using Deep Learning

## Project Overview

This project focuses on recognizing human emotions from speech signals using audio processing and Deep Learning techniques.

## Emotions Classified

- Angry
- Disgust
- Fear
- Happy
- Neutral
- Sad
- Surprise

## Technologies Used

- Python
- Pandas
- NumPy
- Librosa
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

1. Load speech dataset
2. Extract emotion labels
3. Perform data preprocessing
4. Visualize audio waveforms
5. Generate spectrograms
6. Train Deep Learning model
7. Predict emotions from speech

## Features

- Audio signal analysis
- Waveform visualization
- Spectrogram generation
- Emotion classification
